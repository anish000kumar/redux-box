Redux Box · Live Examples

Two end-to-end TypeScript apps that exercise the Redux Box conventions on real public APIs. Both are open in their full source under examples/ in the repository.

CRUD · REST Live

List, create, edit and delete posts against JSONPlaceholder.

  • Optimistic UI with snapshot rollback
  • Normalised byId / allIds state
  • Debounced search via a lazy selector
  • Cross-module saga driving toasts on lifecycle events

Trello · GraphQL Live

Three-column Kanban board on GraphQL Zero, with native HTML5 drag-and-drop.

  • Optimistic create / rename / delete with rollback
  • Move-card saga with same-status short-circuit
  • Per-card pending flag via lazy selector
  • Tiny zero-dep fetch-based GraphQL client
Built with redux-box, TypeScript, redux-saga, immer and reselect.
